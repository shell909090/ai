# AI相关项目

* [all2txt](all2txt/): 从任意格式文件中提取纯文本。支持OCR和ASR。
* [comfyuiapi](comfyuiapi/): 通过代码直接驱动comfyuiapi来生成和处理图片的项目。
* [dkai](dkai/): 通过docker来管理AI环境的项目。
* [elocate](elocate/): 一个面向本地文档的语义搜索工具，类似 `mlocate`，但使用向量引擎进行语义匹配。实测效果不佳，废弃。
* [llm4decompile](llm4decompile/): llm4decompile的使用记录。
* [prompts](prompts/): 我常用的一些提示词。
* [summary](summary/): 自动摘要类，包括自动对纽约时报中文版做摘要的项目。
* [tgbot](tgbot/): 利用/loop指令，把claude code改造成可以通过IM操纵的模式。
* [HITL-proxy](HITL-proxy/): human in the loop工具。允许AI去调用外部API。项目管理token，避免AI直接接触。同时人类二次审批。
